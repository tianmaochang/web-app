Focus Wine Cellars (FWC) 是一家设计和建造定制化酒窖与储酒柜的一流企业，可满足来自全球的私人和商业的需求。
 
从欧洲、亚洲到南美洲... FWC活跃在三个大洲上的多个国家。
 
FWC的主流客户不仅有杰出人物，国家领导人和葡萄酒收藏家，还包括知名酒店品牌，米其林星级餐厅和龙头房地产开发商。
 
您在定制化酒窖方面的全球解决方案合作伙伴。
